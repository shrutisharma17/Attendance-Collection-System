Attendance Collection System
By Shruti Sharma

shrutii1710@gmail.com

  • This is a Face Recognition based Attendance System.

  • It was made using Django framework with the help of Python's OpenCV library.

  • See the next slides for further information!
Instructions:

  • Click on "Next page button" to get to home page.

  • For a new member, click on "Members" and further select "Add member" to enter the details of new member.

  • Make sure to upload your correct name and information.

  • The photograph uploaded should be of good quality and its file name should be the proper name of the person.

  • Follow the next steps for both, new and existing member

Instructions:

  • Now, select "Take attendance to run the scanner to take the attendance."

  • The scanner may take some time to initiate depending on the hardware of the pc.

  • Once the scanner shows the correct name of the file uploaded, press "q" to exit scanner.

  • Note: You can take attendance of a person only once in 24 hours. Morever you can also edit or delete the details via the buttons provided.
